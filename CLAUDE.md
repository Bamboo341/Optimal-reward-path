# CLAUDE.md

## プロジェクト概要
OSM地図上で報酬エッジを手動設定し、距離制約下で報酬最大化経路を探索するローカルツール。
仕様は `docs/SPEC.md` が唯一の正。実装前に必ず全文を読むこと。

## 開発ルール
- SPEC.md の Phase 0 → 4 の順に実装し、各Phaseの受入基準を満たしてから次へ進む
- 各Phase完了時に `pytest` を実行し、全テスト通過を確認する
- 仕様に曖昧さを見つけた場合、SPEC.md「設計判断の根拠」の方針（実装難易度最小・報酬エッジ少数前提）に沿って解釈し、判断内容をコミットメッセージに記録する
- 依存ライブラリは requirements.txt にバージョン固定で追加する
- グラフデータ（*.graphml）と output/ は .gitignore に含める

## 起動・テスト
- 起動: `streamlit run app.py`
- テスト: `pytest tests/`
